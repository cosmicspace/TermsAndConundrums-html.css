MIT License

Definition and details

A short, permissive software license. Basically, you can do whatever you want as long as you include the original copyright and license notice in any copy of the software/source. There are many variations of this license in use.

When this should be used

When someone wants basic license to open source software but wants to retain copyright.

History and shenanigans

The MIT License was originally developed at MIT, hence the name. However, this license is not copyrighted, so the license can be modified. For that reason, there are many variants. The 2 main ones are Ex-Pat and the X11 license. The main difference between the 2 is that the X11 license adds to the original the following: “Except as contained in this notice, the name(s) of the above copyright holders shall not be used in advertising or otherwise to promote the sale, use or other dealings in this Software without prior written authorization.”
