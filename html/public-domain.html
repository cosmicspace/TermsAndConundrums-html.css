Public Domain

Definition and details

Being in the public domain is not a license; rather, it means the material is not copyrighted and no license is needed. An exception to this is the Creative Commons CC0 License and the Unlicense, which deliberately place a work into Public Domain. Practically speaking, though, if a work is in the public domain, it has an all-permissive non-copyleft free software license. Public domain material is compatible with the GNU GPL.

If you want to release your work to the public domain, the Free Software Foundation encourages you to use formal tools to do so. They ask people who make small contributions to GNU to sign a disclaimer form, which is one solution. If you're working on a project that doesn't have formal contribution policies like that, CC0 or the unlicense are good tools that anyone can use. It formally dedicates your work to the public domain, and provides a fallback license for cases where that is not legally possible.

When this should be used

The Apache License 2.0 should be used when you have obtained patents for the sake of others using them freely. This prevents others from similarly patenting your original idea and then disallowing others to use your idea. Using this license allows others to modify, copy and distribute a work or product while giving guarantees to the new authors to use the previous patents.

Length or term of license

The length of time for a work to be in teh Public Domain is in perpetuity, which means forever.

Real world examples

  • CERN httpd, the original World Wide Web daemon developed by Tim Berners-Lee, was in the public domain.
  • SQLite, the most widely-deployed SQL database in the world, is in the public domain.
  • qmail, the second-most popular MTA on the Internet, is in the public domain.
  • C++ Big Integer Library, a library for integer arithmetic, is in the public domain.
  • ALL works published before 1923 are considered in the public domain, since and Copyrights would have expired. There are some exceptions for items published after 1923, depending on their status in terms of Copyright.
  • All works prepared by an officer or employee of the US Government as part of that person's official duties are always in the public domain.
